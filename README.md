# wmsadmin_system
# 权限管理后台系统（.NET Core 9 + Bootstrap 6）
这个是我自己做的一个后台权限管理系统骨架，适合企业后台或者自己练手用。前端我用了 **Bootstrap 6 + 原生 JS + cshtml**，所有静态文件都在本地，没用别的前端框架。后端是 **.NET Core 9 + EF Core + MySQL**。整个项目我拆成了几个层次，便于后续扩展和维护：

- **Core**：实体和接口  
- **Infrastructure**：数据库操作（EF Core）  
- **Application**：业务逻辑  
- **Web**：后台页面和布局  

---

## 主要功能

- **用户管理**：可以加人、删人、改人，还能给用户分配角色  
- **角色管理**：角色可以绑定菜单和按钮权限  
- **菜单管理**：
  - 可以自己创建、编辑、删除菜单  
  - 支持多级嵌套（树形）  
  - 可以设置是内部页面还是外部链接  
- **按钮权限**：  
  - 按钮必须挂在菜单下  
  - 可以控制角色是否能看到或点击  
- **页面权限**：  
  - 如果用户没权限，菜单和页面就完全看不见  

左边菜单树，右边内容区，典型后台管理布局，操作起来很直观。

---

## 项目亮点

- **清晰分层**：后端代码分层，前端纯 cshtml+JS，方便扩展业务模块  
- **易上手**：有完整 CRUD 示例，登录和权限管理功能都在  
- **现代化界面**：简洁、左侧导航、右侧内容区域，符合常见后台风格  
- **可复用**：可以直接拿来做基架，后续业务模块加上去很方便

## 快速开始

1. 克隆仓库  
2. 配置 MySQL 数据库  
3. 修改 `appsettings.json` 里的数据库连接  
4. 用 EF Core 迁移创建表  
5. 启动 Web 项目  
6. 登录试试菜单和权限示例
